# coffeefy

## Description

Coffeefy let you control a Coffee maker via Internet

## Installation
- clone repository
- navigate to ./node
- run **npm install** to install the dependencies
- To start the webserver, run: **nodemon app**
- open **http://localhost:3000** in browser
